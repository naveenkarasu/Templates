## Using a Template

### Static templates (21 templates)

Just open `index.html` in a browser. No build step needed.

### Node.js templates (4 templates)

These templates require `npm install` before running:

| Template | Framework | Commands |
|----------|-----------|----------|
| [3d-keyboard-skills](templates/3d-keyboard-skills) | Next.js | `pnpm install && pnpm dev` |
| [block-reveal-transition](templates/block-reveal-transition) | Next.js | `npm install && npm run dev` |
| [turbulent-lens-hover](templates/turbulent-lens-hover) | Vite | `npm install && npm run dev` |
| [will-you-be-my-valentine](templates/will-you-be-my-valentine) | Next.js | `pnpm install && pnpm dev` |
